Client-side usage

Get the client-side version of AbsurdJS from here and add it to your page.

<script src="absurd.min.js"></script>

Initialize Absurd and use the API.

var api = Absurd();
api.add({
	body: {
		marginTop: "20px",
		p: {
			color: "#000"
		}
	}
});

Compile and get the CSS.

api.compile(function(err, css) {
	// use the compiled css
});
